# AllPay

## Stablecoin-native invoicing and payments built on Arc

AllPay is a modern invoicing and payment platform designed for the internet economy.

The platform enables freelancers, creators, startups, agencies, and global teams to create invoices, share payment links, and receive instant stablecoin payments without relying on traditional banking infrastructure.

By leveraging Arc's real-time settlement infrastructure and stablecoin-native architecture, AllPay aims to make global payments as seamless as sending a message online.

---

## Problem

Traditional invoicing systems suffer from several limitations:

* Banking hours delay settlement.
* International transfers can take days.
* Payment fees are often unpredictable.
* Cross-border transactions involve multiple intermediaries.
* Currency conversion creates additional friction.

Internet businesses need financial infrastructure that operates at internet speed.

---

## Solution

AllPay provides a simple workflow:

1. Create an invoice.
2. Generate a payment link.
3. Share the link with a customer.
4. Receive USDC payment instantly on-chain.
5. Track invoice status in real time.
